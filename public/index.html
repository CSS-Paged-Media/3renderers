3renderers.dev

Free HTML-to-PDF rendering service using the open-source tools Paged.js, WeasyPrint, and Vivliostyle. 		Version v0.0.1
Updated 2025-10-23
Author Andreas Zettl Postman Collection

The service allows you to create PDFs from your HTML and CSS source free of cost.

Please be aware that there is no guarantee that this service will be available or that the current rate limits and monthly traffic restrictions will stay the same.

Once the monthly max. traffic is reached, the service will return an error message. On the first of each month, the limit is reset, and you will be able to render more PDFs.

Also, please note there is no paid version of this service to get rid of the restrictions, but there are many great ones out there (for example, EuroPDF).

Quick Start

Fast test with curl (sync rendering):

curl -X POST http://localhost/api/render \
      -H "Content-Type: application/json" \
      -d '{"html":"<h1>Hi</h1>","options":{"renderer":"weasyprint","sync":"true"}}' --output result.pdf

For more advanced usage, please refer to the API documentation.
